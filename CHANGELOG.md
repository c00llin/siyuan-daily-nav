# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-01-18

### Added
- Navigate to previous day with ⌥⌘← (Option+Command+Left Arrow)
- Navigate to next day with ⌥⌘→ (Option+Command+Right Arrow)
- Auto-create daily notes if they don't exist
- Respect notebook's daily note format configuration
- Smart notebook detection (uses same notebook when in a daily note)
- Configurable fallback notebook setting
- Custom attribute `custom-dailynote-YYYYMMDD` added to created daily notes
- Respect SiYuan's "open in current tab" preference
- Silent navigation (only shows message when creating new notes)
